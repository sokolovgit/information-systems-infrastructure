Висновок:

У ході виконання сьомого модуля було опановано підхід "Інфраструктура як код" через використання шаблонів Azure Resource Manager. Було вивчено декларативний метод опису хмарної інфраструктури, що дозволяє автоматизувати процеси створення та керування ресурсами. Під час практичної частини було розроблено JSON-шаблон для створення облікового запису сховища з використанням параметризації для гнучкого налаштування імен та типів ресурсів. Також було налаштовано секцію вихідних параметрів для автоматичного отримання технічної інформації про створені об'єкти.
